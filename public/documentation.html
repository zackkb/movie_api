My Movies App

Web application will provide users with information about movies, directors, and genres.
Users will be able to sign up, create a favorites list, and update personal information.



URL requests and endpoints


Request URL HTTP Method Query parameters Request Response
Return a list of ALL movies to the user /movies/[id] GET :id None A JSON object holding data about all movies
Return data about a single movie by title to the user /movies/[title] GET :title none A JSON object holding data about a single movie. Example:
                    
          {
            Title: '', 
            Description: '',
            Year: '', 
            Genre: {
              name: '',
              description: ''
            } 
            Director: {
              name: '', 
              birth: '',
              death: '',
              bio: ''
            },
            Actors: {
        
            }
            ImageURL: '',
            Featured: ''
          }
Return data about a genre (description) by name/title (e.g., “Comedy”) /movies/genres/[genre] GET :genre none A JSON object holding data about a genre. Example:
                    
          { 
            name: 'Comedy', 
            description: '', 
          }
Return data about a director by name /movies/directors/[name] GET :name None A JSON object holding data about the selected director. Example: 
              
    { 
      name: 'Hiroyuki Okiura', 
      birth: '1966', 
      death: '-',
      bio: 'Hiroyuki Okiura is a Japanese animation director and animator working for Production I.G.'
    }
Allow new users to register /users POST None A JSON object holding data about user being added, structured like:
                    
          { 
            "username": "Zach", 
            "favorites": [
              ...
            ] 
          }
JSON object holding data about added user including ID. Ex: 
            
  { 
    id: '1',
    "username": "Zach", 
    "favorites": [
      ...
    ] 
  }
Allow users to update their user info /users/[id] PUT :id A JSON object holding the new username. Ex: 
            
  { 
    "name": "Zach"
  }
JSON object with the updated user. Ex: 
            
  { 
    id: '1',
    name: 'Zach', 
  }
Allow users to add a movie to their list of favorites /users/[id]/[movie] POST :id, :movie None Message indicating the movie was added
Allow users to remove a movie from their list of favorites /users/[id]/[movie] DELETE :id, :movie None Message indicating the movie was deleted
Deletes a user /users/[id] DELETE :id None Messege stating user has been deleted
